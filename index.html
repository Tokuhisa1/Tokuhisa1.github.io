Temperature Conversion

This ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding temperature is in Fahrenheit. 

def cel_2_fah()
  print 'Enter degrees in Celsius: '
  celsius = gets.to_i
  fahrenheit = celsius * 9 / 5 + 32

  puts "\nThe temperature is " + fahrenheit.to_s + ' degrees Fahrenheit.'
end

def get_ordinal(num)
  if num < 10 && num % 10 == 1
    return num.to_s + 'st'
  elsif num < 10 && num % 10 == 2
    return num.to_s + 'nd'
  elsif num < 10 && num % 10 == 3
    return num.to_s + 'rd'
  else
    return num.to_s + 'th'
  end
end

Foobar

This ruby program is an adaptation of a classic technical interview problem which displays sequences of the Foobar pattern. 

def foo_bar()
  print 'How many items do you want to see? '
  items = gets.to_i
  n = 1

  while n <= items
    if n % 15 == 0
      puts "\nFoobar"
    elsif n % 5 == 0
      puts "\nBar"
    elsif n % 3 == 0
      puts "\nFoo"
    else
      puts "\n" + n.to_s
    end

    n += 1
  end
end
